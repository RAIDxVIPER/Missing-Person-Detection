# 👤 Missing Person Detection

## 📌 Overview
Missing Person Detection is an AI-powered system that helps locate missing individuals using facial recognition technology. This project leverages **OpenCV**, **Deep Learning**, and **Machine Learning** to analyze images and videos to identify missing persons. 🚀

## 🎯 Features
✅ **Facial Recognition:** Detects and matches missing persons from a database.  
✅ **Live Camera Integration:** Scans real-time video footage for potential matches. 📹  
✅ **Database Management:** Stores missing persons' details for comparison. 📂  
✅ **Alert System:** Notifies authorities when a match is found. ⚠️  
✅ **Scalable:** Can be integrated with CCTV, social media, and government databases. 🌍  

## 🛠️ Tech Stack
- **Python 🐍**
- **OpenCV 👀**
- **TensorFlow/Keras 🧠**
- **Dlib**
- **Flask (for Web API) 🌐**
- **SQL/NoSQL Database 💾**

## 🔍 How It Works
1. **Image Input**: Upload an image of the missing person. 📸  
2. **Feature Extraction**: The system extracts facial features using deep learning. 🧠  
3. **Database Matching**: Compares the extracted features with stored data. 🔎  
4. **Detection & Alert**: If a match is found, an alert is generated. 🚨
